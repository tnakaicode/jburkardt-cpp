mitchell_09
Wave Front Problem

mitchell_09, Mitchell's "wave front" problem on the [0,+1]x[0,+1] square. The parameters ALPHA, XC, YC, and R0 may be varied.

  • ALPHA = 20, XC = -0.05, YC = -0.05, R0 = 0.7.
  • ALPHA = 1000, XC = -0.05, YC = -0.05, R0 = 0.7.
  • ALPHA = 1000, XC = 1.5, YC = 0.25, R0 = 0.92..
  • ALPHA = 50, XC = 0.5, YC = 0.5, R0 = 0.25.
