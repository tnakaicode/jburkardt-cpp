mitchell_07
Boundary Line Singularity Problem

mitchell_07, Mitchell's "boundary line singularity" problem on the [0,+1]x[0,+1] square. There is a singularity in the right hand side function F(X,Y), at the left boundary. We use the parameter value ALPHA = 0.6.
